## 📌 Project Overview

The **Loan Data Analysis Dashboard** is a comprehensive Business Intelligence solution developed using **Microsoft Power BI** to analyze customer loan data, financial risk, loan defaults, and applicant demographics. The dashboard enables financial institutions to monitor lending performance, identify risk patterns, and gain actionable insights for better decision-making.

This project demonstrates an end-to-end data analytics workflow, including data cleaning, data modeling, DAX calculations, and interactive dashboard design.

---

## 🎯 Business Problem

Financial institutions process thousands of loan applications every year. Understanding customer profiles, loan performance, and default behavior is essential for reducing financial risk and improving lending strategies.

This dashboard helps answer key business questions such as:

- Which customer groups borrow the most?
- How have loan amounts changed over time?
- Which employment types have higher default rates?
- How does credit score influence loan approval and loan amount?
- What demographic factors affect lending behavior?
- Which loan purposes contribute the highest loan amounts?
- How are financial risks changing year-over-year?
